A lite Weight Plugin that helps you, Easily showcase your Books and other items in your WordPress Website in Post, Page, Widget Area using shortCode. 

**[Live Demo ](http://showcase.rama.bplugins.com/)**

= shortCode =
Follow the steps below to generate a shortCode.

1. Go to -> Showcase Viewer from Dashboard menu -> Add New Showcase.
2. Configure the viewer from Showcase Settings with your preference.
3. Copy Generated ShortCode and paste it where you like to embed the Showcase.
4. Enjoy Showcase!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want to show showcase in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates
